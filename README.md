# Підрахунок очок

Mobile-first PWA для підрахунку очок у карткових іграх.

MVP сфокусований на швидкому веденні партії з iPhone: шаблони ігор, колоди карт, гравці, раунди, загальний рахунок та історія завершених партій.

## Поточний MVP

- Шаблони ігор з правилами.
- Колоди карт із власними очками.
- Preset `Брідж` з колодою на 36 карт.
- Партії з гравцями.
- Ввід раундів вручну або через вибір карт.
- Додатні та відʼємні очки через `+ / -` перемикач.
- Множник раунду.
- Перевірка ліміту очок як перемоги або програшу.
- Історія завершених партій.
- Експорт та імпорт локальних даних у JSON.
- PWA manifest та mobile-first layout для iPhone.

## Технології

- Next.js
- React
- TypeScript
- Tailwind CSS
- LocalStorage для MVP-даних

## Запуск локально

```bash
npm install
npm run dev
```

Відкрити на Mac:

```text
http://localhost:3000
```

Відкрити на iPhone в тій самій Wi-Fi мережі:

```text
http://<mac-local-ip>:3000
```

Наприклад:

```text
http://192.168.1.7:3000
```

## Перевірки

```bash
npm run typecheck
npm run build
```

## Дані

Зараз дані зберігаються локально в браузері через `localStorage`.

У вкладці `Дані` можна:

- експортувати JSON backup;
- імпортувати JSON backup;
- скинути дані до preset-шаблону `Брідж`;
- очистити всі локальні дані.

Імпорт перевіряє структуру JSON перед записом у сховище.
